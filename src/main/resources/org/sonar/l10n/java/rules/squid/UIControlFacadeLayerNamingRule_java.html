com.sinosoft.claim.ui.control.facade 包下的类名必须以 UI 开头，以 Facade 结尾。

不合规范代码示例

package com.sinosoft.claim.ui.control.facade;

public class AuditSearchFacade  extends Action{ // Noncompliant

}

合规代码示例

package com.sinosoft.claim.ui.control.facade;

public class UIAuditSearchFacade  extends Action{ // Compliant

}
