com.sinosoft.claim.ui.control.action 包下的类名必须以 UI 开头，以 Action 结尾。

不合规范代码示例

package com.sinosoft.claim.ui.control.action;

public class AuditSearchAction  extends Action{ // Noncompliant

}

合规代码示例

package com.sinosoft.claim.ui.control.action;

public class UIAuditSearchAction  extends Action{ // Compliant

}
