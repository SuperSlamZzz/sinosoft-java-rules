Dto类属性需用private修饰并初始化为null。

不合规范代码示例

package com.sinosoft.claim.dto.claimdamages;

public class DamagesItemKind {
    public String insurecode = "011100001010101"; // Noncompliant
}

合规代码示例

package com.sinosoft.claim.dto.claimdamages;

public class DamagesItemKind {
    public String insurecode = null;  // Compliant
}
